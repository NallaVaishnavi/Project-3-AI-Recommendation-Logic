# Project-3-AI-Recommendation-Logic

## Project Overview

This project is a simple AI Recommendation System developed using Python. The system recommends books based on user interests using similarity matching logic.

Users enter their interests, and the system compares them with predefined book tags. A match score is calculated for each book, and the books with the highest scores are recommended.

## Objective

To create a recommendation system that matches user preferences with item attributes and provides relevant recommendations.

## Features

* Accepts user interests as input
* Uses similarity-based matching logic
* Calculates match scores
* Displays personalized book recommendations
* Supports multiple interests
* Handles cases where no matching recommendations are found

## Technologies Used

* Python
* Dictionaries
* Loops
* Conditional Statements
* Sorting

## Dataset

The system contains the following books:

* Atomic Habits
* Rich Dad Poor Dad
* The Alchemist
* Deep Work
* Think and Grow Rich

Each book is associated with tags such as motivation, productivity, self-help, finance, money, business, adventure, fiction, focus, career, and success.

## How It Works

1. The user enters interests separated by commas.
2. The program converts the input into a list.
3. User interests are compared with the tags of each book.
4. A match score is calculated based on matching interests.
5. Books are sorted according to their scores.
6. The most relevant books are recommended.

## Sample Input

finance, motivation

## Sample Output

Recommended Books:
